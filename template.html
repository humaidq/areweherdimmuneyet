Explanation

Assuming R0 of COVID-19 is 3 when no precautions are taken. We can calculate herd immunity threshold using the formula: 1-(1/R0). In this case it is 0.67 (67%).

Since the number of the people who took two doses are not provided, and knowing each person has to take two doses. We chose 140 doses per 100 people (~70%) as the goal.

This estimate is predicted using linear regression, expect it to change frequently. Remember, this is just an estimation on an arbitrary goal.
